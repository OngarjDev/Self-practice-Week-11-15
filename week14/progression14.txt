ซึ่งเป็น Format แบบ ISO DATETIME DATABASE สังเกตจาก 
ดังนั้น Type จะเป็น string แต่ รูปแบบการเขียนเป็น UTC

const declaredAt = new Date(data.updatedAt);
เมื่อเราได้ string จาก Backend เราก็เอา string ไปแปลงเป็น obj new Date()

const timeZone = Intl.DateTimeFormat().resolvedOptions().timeZone;
