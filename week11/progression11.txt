สัปดาห์นี้อาจารย์ให้ฝึกการใช้ async await 
await ที่ผมเข้าใจเหมือนเป็นการรอ ถึงจะไปทำบรรทัดด้านล่างได้ 
โดยที่ ต้องขึ้นเป็น async function เพื่อที่จะใช้ await ได้ 2 ตัวนี้มาคู่กัน
ผมได้เรียนรู้ ว่า promise มี 
3 state 
fullfill คือรอให้ api ตอบกลับมา
then คือ ทำต่อเมื่อสำเร็จ
catch คือ ทำเมื่อเกิดข้อผิดพลาด
